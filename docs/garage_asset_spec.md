# Garage Avatar Asset Spec

- **Format:** Transparent PNG
- **Source resolution:** Provide one high-res source at 1024x1024 (recommended)
- **Safe area:** Artwork must fit inside a circle using ~88% of the diameter to avoid clipping in round profile icons
- **Naming:** `avatar_<YYYY_MM>_<name>.png`

## Required Metadata
- `id`
- `name`
- `monthLabel`
- `purchaseType`
- `prices`
- `dates`
- `published`

## Preview Checklist
- Legible at 24px (leaderboard)
- Legible at 40px (tab icon)
- Legible at 96px (profile header)

## Content Rules
- No brand logos or copyrighted marks
